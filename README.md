# Exodus Church — План Победы

Финальная проверенная версия.

## Главное исправление
Каждая кнопка получает собственный прямой адрес на bibleplan.ru:

- Псалом 10 → `/bible/psalms/10`
- Матфея 10 → `/bible/matthew/10`
- 2 Паралипоменон 22 → `/bible/2chron/22`

## Что проверено автоматически
- JavaScript-синтаксис файлов;
- 66 книг Библии;
- первая и последняя глава каждой книги;
- отклонение несуществующих номеров глав;
- 365 календарных адресов;
- отдельные ссылки сегодняшнего примера.
