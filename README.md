# Exodus Church — План Победы

Финальная версия с точным официальным календарём на 365 дней.

## Как работает
- `days.json` содержит все 365 официальных ежедневных адресов bibleplan.ru.
- `/api/today.js` загружает официальный день и извлекает названия всех глав.
- Каждая кнопка открывает официальную страницу нужного календарного дня.
- Никаких вручную придуманных чтений и неправильных book-slug ссылок нет.
