## Установка
1. Удалите старые файлы репозитория.
2. Загрузите все файлы из этой папки в корень GitHub-репозитория.
3. В Vercel: Framework Preset = Other.
4. Build Command и Output Directory оставьте пустыми.
5. Сделайте Redeploy.
6. На телефоне удалите старую иконку приложения и добавьте её заново, чтобы очистить старый кеш.
