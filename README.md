# Exodus Church — План Победы

Новая версия приложения на Next.js.

## Как установить

1. Удалите старые файлы из репозитория GitHub.
2. Загрузите все файлы и папки из этого архива в корень репозитория.
3. Vercel автоматически определит Next.js и выполнит новый deployment.
4. Дождитесь статуса Ready.

## Локальный запуск

```bash
npm install
npm run dev
```

## Важно

- Данные плана находятся в `src/data/plan.js`.
- Логотип находится в `public/logo.jpeg`.
- OneSignal использует существующий App ID проекта.
